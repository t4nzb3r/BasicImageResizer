Name: NoggoNoggoResizer
Author: Tim Schneider

Description: This is a small, but quiet high quality image converter. The width and the height can be scaled individually by specified factor.

How To Use: Press on the button with "..." on it and select your file. The current width and height gets shown. Now enter a faktor for resizing (smaller 1: shrinking, bigger 1: expand). Click on resize for saving. The new file name is: [old filename]_[new width]x[new height].[old extension]. It is located in the same folder as the starting picture.

Some Facts:
	- Compositing Mode: Source Copy
	- Compositing Quality: High
	- Interpolation Mode: Nearest neighbor
	- Smoothing Mode: None
	- Pixel Offset Mode: High Quality
